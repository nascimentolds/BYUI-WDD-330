Notes for week 9

The Window Object

  • The window object is the global object; contains global variables and methods as its properties
  • this in the global scope refers to the global object
  • alert(), confirm(), and prompt() are methods of the window object
  • The Navigator object contains information about the browser. Access with window.navigator
  • window.location contains information about the URL of the current page
  • Use window.history to access info about history in the current session

HTML5 APIs

data- property

  • data- attribute embeds data with an element that is ignored by the browser
  • Access an element's data- properties with element.dataset.myProperty

Web Storage API

  • Web Storage API (localStorage) lets you save key-value pairs. Lots of advantages over cookies.
  • localStorage.name = 'Falco';
  • localStorage.removeItem('name'); or delete localStorage.name
  • localStorage.clear() removes everything stored in local storage (for that domain)
